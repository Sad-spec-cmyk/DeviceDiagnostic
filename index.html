Windows diagnostic utility

Один інтерфейс для швидкої перевірки стану комп'ютера

Device Diagnostic показує системну інформацію, графіки навантаження CPU, RAM і GPU, запускає мережевий та дисковий тести і дозволяє експортувати результати в JSON. Тепер програму можна скачати як готовий Windows .exe, а не як сирий Python-код.

Основні можливості

Система

Показує ОС, кількість ядер процесора, обсяг оперативної пам'яті та інформацію про GPU.

Продуктивність

Оновлює графіки завантаження CPU, RAM, GPU, VRAM і температури у майже реальному часі.

Мережа

Запускає speed test і показує ping, швидкість завантаження та віддачі.

Накопичувач

Виконує базовий тест запису та читання тимчасового файлу, щоб оцінити швидкість диска.

Експорт

Зберігає короткий діагностичний звіт у файл JSON для подальшого використання.

Інтерфейс

Темна техно-стилістика з акцентами, зручною для локальної утиліти моніторингу.
