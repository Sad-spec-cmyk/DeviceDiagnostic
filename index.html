What the program can do

Main Features

System

Shows OS, CPU cores, RAM size, and GPU information.

Performance

Updates CPU, RAM, GPU, VRAM, and temperature graphs in near real-time.

Network

Runs a speed test and shows ping, download, and upload speeds.

Storage

Performs a basic read and write test on a temporary file to assess disk speed.

Export

Saves a short diagnostic report to a JSON file for further use.

Interface

Dark techno-style interface with accents, convenient for a local monitoring utility.
